# Roadmap

This is a starter roadmap you can expand week by week.

## Phase 1 — Fundamentals (Weeks 1–2)
- Cloud basics + core concepts (identity, networking, storage)
- GitHub + documentation habits

## Phase 2 — Core platform skills (Weeks 3–6)
- Networking basics, access control, monitoring basics
- Build 1 small “platform” project and document it

## Phase 3 — IaC + CI/CD (Weeks 7–10)
- Terraform fundamentals, remote state
- CI pipeline: fmt/validate/plan

## Phase 4 — Portfolio + hiring (Weeks 11–12)
- 2–3 projects on GitHub
- CV/LinkedIn + interview practice
